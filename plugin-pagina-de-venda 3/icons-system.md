## Guia de Uso nos Skills

### No **copywriter** SKILL
- Use ícones SVG apenas em descrições/exemplos (não na copy gerada)
- Exemplo: "✓ Benefício 1" → Use Circle Bullet ou Checkmark em documentação

### No **designer-html** SKILL
- Defina uma **cor padrão para ícones** no DESIGN_TOKENS: `"icon_color": "{cor_primaria}"`
- Escolha 3-5 ícones principais baseado no nicho
- Exemplo para **educação**: Checkmark, Arrow Right, Trending Up, Heart
- Exemplo para **fitness**: Zap, Heart, Trending Up, Briefcase (pro coaching)

### No **construtor-bloco1** SKILL
- Use Checkmark para benefícios (seção "CTA Text")
- Use Heart para "Para quem é" (emoção)
- Use X para "Para quem não é" (eliminação)

### No **construtor-bloco2** SKILL
- Use Checkmark para entregáveis
- Use Shield para garantia
- Use Heart para depoimentos emocionais

### No **construtor-bloco3** SKILL
- Use User/Person para autoridade + foto
- Use Question Mark para FAQ
- Use Shield para garantia/suporte

### No **construtor-bloco4** SKILL
- Use Trending Up para argumentos incontestáveis (dados)
- Use Heart para dores & desejos (emoção)
- Use User/Person para "sobre o autor"
- Use Arrow Right para comparação de etapas

## Implementação CSS

### Classe Base (adicionar ao construtor-bloco1)
```css
.icon {
  width: 24px;
  height: 24px;
  display: inline-block;
  vertical-align: middle;
  margin-right: 8px;
  color: var(--primary);
}

.icon.small { width: 16px; height: 16px; margin-right: 4px; }
.icon.large { width: 32px; height: 32px; margin-right: 12px; }
.icon.accent { color: var(--accent); }
.icon.text-dark { color: var(--text-dark); }
```

### Uso em HTML
```html
<li>
  <svg class="icon" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <polyline points="20 6 9 17 4 12"></polyline>
  </svg>
  Benefício importante
</li>
```

## Regras Críticas

1. **Nada de emojis** (🎯 🚀 ⚡ 📈 🎥 ✅ ❌ etc)
2. **Todos os ícones são SVG inline** (não imagens .png/.jpg)
3. **Ícones herdam cor do texto** via `currentColor` ou `color: var(--primary)`
4. **Geometria pura**: linhas, círculos, polígonos, sem detalhes
5. **Tamanho padrão**: 24px (redimensionável via CSS)
6. **Stroke width**: 2 para ícones com outline; "fill" para ícones sólidos
7. **Consistência**: Usar os mesmos ícones em toda a página
